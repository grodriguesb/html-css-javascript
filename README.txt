Repositório dedicado ao desenvolvimento de mini projetos e experimentos práticos com HTML, CSS e JavaScript, voltados ao aprimoramento contínuo das habilidades em front-end.

Os projetos abordam conceitos como estruturação semântica, estilização responsiva, manipulação do DOM, eventos e lógica em JavaScript, organizados de forma modular para facilitar manutenção, aprendizado e evolução técnica.
